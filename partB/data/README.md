# Dataset Description

## Synthetic 2D Point Set Registration Dataset

This dataset is generated programmatically in `task_21.ipynb`. No manual download is required.

### How the dataset is obtained
- **Model point set**: ~150 points sampled from a 2D closed curve (circle + sinusoidal perturbation) to create a non-trivial shape.
- **Scene point set**: The model points are transformed using a known non-rigid deformation (thin-plate spline warping), with added Gaussian noise (σ=0.02) and ~20 random outlier points.

### How it is used
- `task_21.ipynb`: Generates and visualizes the model and scene point sets.
- `task_22.ipynb`: Applies the GMM-based registration method to align the model to the scene.
- `task_23.ipynb`: Evaluates registration quality using Mean Squared Error (MSE).
- `task_31.ipynb`: Uses the same dataset for ablation experiments.
- `task_32.ipynb`: Generates a modified dataset to demonstrate failure modes.

### Files
- `model.txt`: 2D coordinates of the model point set (generated by `task_21.ipynb`)
- `scene.txt`: 2D coordinates of the scene point set (generated by `task_21.ipynb`)
- `scene_clean.txt`: Ground-truth deformed model without noise/outliers (for evaluation)
